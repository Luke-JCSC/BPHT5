HTML Templates

While we want to be proud of 100% HTML compliance and buzzword bingo, but for now the focus has been on functionality and getting the components wrapped up in a manor I can release the tools, used to generate these templates. The possibilities are endless with customization. You will see many different use cases providing proper use of elements and providing an elegant UI. There Are different templates, and not all the html files utilize the same appearance. We are working on breaking up the UI points into themes for these templates, but it is a means to provide the end user with the tools needed to ignore the web design and focus on their project or documentation efforts.
